# Write-a-Python-program-to-find-minimum-element-from-a-list-of-elements-
L=[2,58,95,999,65,32,15,1,7,45]
print(L)
m=min(L)
print("The minimum elements is:",m)
print("Index of minimum element is:",L.index(m))
 

OUTPUT:

[2, 58, 95, 999, 65, 32, 15, 1, 7, 45]
The minimum elements is: 1
Index of minimum element is: 7
